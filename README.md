
# House Price Prediction using Linear Regression

## Project Overview

This project implements a Linear Regression model using the California Housing Dataset from Scikit-Learn.

## Dataset

California Housing Dataset

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Workflow

1. Data Loading
2. Exploratory Data Analysis (EDA)
3. Data Preprocessing
4. Train-Test Split
5. Linear Regression Model Training
6. Model Evaluation
7. Visualization

## Model Performance

* MAE: 0.533
* RMSE: 0.746
* R² Score: 0.576

## Visualizations

* Correlation Heatmap
* Actual vs Predicted Plot
* Residual Plot
